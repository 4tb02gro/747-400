# Installation Instructions:

1: Unzip "Boeing 747-400 FMOD"

2: Drag and drop "fmod" and "sound" folders into default (Boeing 747-400) or mSparks (747-400-master/Sparky 744) 747 main folder

3: enjoy your flight!
